## 1. 项目定位

一个兼具前沿科技质感与深厚人文关怀的**「全球语言谱系赛博全息馆」**。通过可视化技术，将人类约 5000 年的语言演化史、空间迁徙地图、以及濒危文化反思，打包成一个"一分钟开箱即食"的数字档案馆。

核心体验三句话：
- **看清历史血缘**：用星系图谱打破"芬兰语和英语是亲戚"这类地理偏见。
- **直观感知文明**：点击语言即触发「地理空间 + 硬核文化」联动闭环。
- **守护濒危火种**：用克制的「文明留白」排版，为即将消失的语言拉起数字防线。

## 2. 三大核心功能规格

### 2.1 赛博星系图谱（核心）
| 项 | 规格 |
|---|---|
| 形态 | 神经网络般的**动态引力星系**，支持自由缩放/拖拽 |
| 节点 | 每种语言一个节点，大小可表示使用人口/影响力 |
| 着色 | 不同**霓虹色块**代表不同语系，一眼看清亲缘关系 |
| 关系 | 边表示语系/亲属关系；无共同祖先的语种在图谱上天然"疏远" |
| 反偏见 | 地理上相近但语系无关的语言（如芬兰语↔英语）在图谱中明显分离 |
| 验收点 | 缩放/拖拽流畅；色块区分清晰；点开任意节点可进入 2.2 闭环 |

### 2.2 空间与趣味「三位一体」闭环
点击任意语言 → 触发以下联动：
1. **地理空间**：右侧**赛博全息点阵世界地图**定向亮起该语言当前/历史上的空间影响力坐标。
2. **硬核文化**：右侧抽屉滑出精炼内容——
   - 「冷知识暴击」（最反直觉的事实）
   - 「发音密码」（魔性语法/独特音系）
   - 「语言梗」（地缘历史故事，网感文案）
3. **联动反馈**：星系图节点高亮、地图点阵点亮、抽屉滑出三者同步、丝滑。

### 2.3 暮色余烬集群（濒危语言）
| 项 | 规格 |
|---|---|
| 视觉 | 由「暮色余晖红」组成的独立濒危语言星云 |
| 动效 | 节点如将燃尽的炭火，**微弱、缓慢的呼吸消逝动效** |
| 排版 | 极具克制与厚重感的「文明留白」版式 |
| 意图 | 展示即将在地球上蒸发的古老神话世界，唤起守护意识 |
| 验收点 | 濒危节点与其他节点视觉区分明确；呼吸动效不刺眼但可感知 |

## 3. 视觉与交互语言（设计基调）

| 维度 | 定义 |
|---|---|
| 底色 | 深空黑 / 极暗蓝紫，营造"全息馆"沉浸感 |
| 主视觉 | 霓虹色块（青/品红/黄绿等分语系），星点粒子 |
| 地图 | **点阵（dot-matrix）世界地图**，非写实轮廓，契合"赛博全息"设定 |
| 动效 | 丝滑联动、缓动过渡；濒危节点用 slow breathing |
| 文案 | 网感、通俗、幽默（"冷知识暴击""语言梗"） |
| 排版 | 常规区信息密度高；濒危区「文明留白」留白克制 |

      "family": "endangered",
      "endangered": true,
      "speakers": 10,
      "coords": [142.0, 43.0],
      "trivia": { "fact": "...", "phonetics": "...", "meme": "..." }

## 5. 技术选型建议（路径 B）

| 层 | 推荐 | 理由 |
|---|---|---|
| 构建 | **Vite + 原生 JS / 或 React+TS** | 现代、可维护、易加新功能 |
| 星系图 | **D3.js force simulation** | 力导向星系、缩放拖拽成熟方案 |
| 点阵地图 | **自绘点阵网格 + 经纬度映射** | 契合"赛博全息"设定，零 GeoJSON 依赖、自包含 |
| 托管 | 静态站点（任意平台） | 彻底脱离造物云锁定 |
| 国际化 | i18n 方案（如 i18next） | 支撑"多语言/国际化"新功能 |

> 当前原型用**单文件 HTML + D3 CDN** 快速验证交互闭环，后续可平滑升级为框架工程。
